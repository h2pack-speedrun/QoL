# QoL

> Practical speedrun quality-of-life options for cleaner menus, faster resets, and better KBM handling.

Part of the [Speedrun modpack](https://github.com/h2pack-speedrun/speedrun-modpack).

## What It Does

QoL bundles a small set of convenience features that smooth out menu handling, cutscene flow, and keyboard-and-mouse use during speedrun attempts.

The current module includes:

- `KBM Escape Fix`
  Makes Escape work correctly during boon/pom selection, Hex selection, Path of Stars, and death sequences. YOU CAN NOW GIVE UP WHEN DYING IF YOU ARE KBM PLAYER.
- `Always Show Location`
  Makes the current location always visible.
- `Skip Death Cutscene`
  Skips the death cutscene and returns you to the main menu faster while still showing the death screen.
- `Auto Skip Dialogue`
  Automatically skips routine dialogue prompts during gameplay while leaving important conversation types alone.
- `Skip Run End Cutscene`
  Skips the end-of-run cutscene and returns you to the main menu faster while still showing the victory screen.
- `Spawn in Training Grounds`
  Starts you in the Training Grounds instead of the usual spawn flow.
- `Arcana & Fear on Victory Screen`
  Adds Arcana and Fear information to the victory screen.

Use this module when you want cleaner attempt flow and fewer menu/input friction points without changing core combat routing.

## Installation

Install via [r2modman](https://thunderstore.io/c/hades-ii/) or manually place in your `ReturnOfModding/plugins` folder.

This module is usually installed as part of the full [Speedrun modpack](https://github.com/h2pack-speedrun/speedrun-modpack), where it appears in the shared Speedrun UI with the other speedrun-focused modules.
